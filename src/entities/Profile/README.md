## Сущность Profile

Описание - сущность профиля содержит в себе все данные о профиле:

1)  id?: string
2)  first?: string
3)  lastname?: string
4)  age?: number
5)  currency?: Currency
6)  country?: Country
7)  city?: string
8)  username?: string
9)  avatar?: string
